# Kapi Clicker

- Live build: [https://kivaball.github.io/kapi-clicker/](https://kivaball.github.io/kapi-clicker/)
- Repo: [https://github.com/KivaBall/kapi-clicker](https://github.com/KivaBall/kapi-clicker)

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

Deploys are handled by GitHub Pages. Any push to `main` triggers the workflow that runs `npm run build` and publishes the contents of `dist/`.
